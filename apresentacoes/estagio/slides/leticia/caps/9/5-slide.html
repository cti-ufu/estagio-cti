4. A criação de um objeto
. Ocorre na etapa "Musico c = new Musico();".
. A palavra new chama um construtor.
. Toda classe criada terá um construtor.
. Caso não seja criado, o compilador inserirá um padrão, sem argumentos.

Construindo um objeto Musico:
	
	public class Musico{ 
		public Musico {  // nome igual ao da classe e sem tipo de retorno.
			System.out.println("Cantar");  // código do construtor.
	}
	public class UseMusico{
		public static void main(Strig[] args){
			Musico c = new Musico(); // chamará o contrutor músico.
	}
	}
. O melhor lugar para inserir o código de inicialização é no construtor. E tudo que é preciso é criar um construtor com argumentos, porém, se preferir tornar a criação do objeo mais fácil, forneça um construtor sem argumentos.
