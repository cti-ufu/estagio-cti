5. Relação das superclasses e a herança com os construtores
. A função dos construtores da superclasse é dar vida a um objeto, e toda superclasse possue um construtor.
. Superconstrutor: são executados para construir as partes do objeto referentes às superclasse.
. Todas as variáveis de instância de qualquer classe da árvore de herança precisam que ser declaradas e inicializadas.
. Encadeamento de Construtores: 
	
	public class Musico{
		public Musico(){
			System.out.println("Criando Musico.");
		}
	}	
	public class Cantor extends Musico{
		public Cantor(){
			System.out.println("Criando Cantor.");	
		}                                           
	}                                               
	public class UseMusico{							
		public static void main(String[] args){
			System.out.println("Iniciando...");
			Cantor c = new Cantor();
		}
	}
OUTPUT:
Iniciando...
Criando Musico.
Criando Cantor.
