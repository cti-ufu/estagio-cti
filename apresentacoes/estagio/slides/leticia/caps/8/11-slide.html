5. Interface

. As interfaces tornam todos os métodos abstratos.
. Para definir uma interface:
	
	   public interface GerenteVendas{...}
. Para implementá-la:
	
	   public class Pessoa extends Empregado implements GerenteVendas{...}
. Criar uma interface:
	
	   public interface GerenteVendas{
	   	public abstract void gerenciarvendas();
	}
