3. Classe Object

.Métodos da Classe:

Fonte:Use a Cabeça! JAVA
.equals(): informa se os dois objetos são iguais.
.getClass(): retorna a classe em que o objeto foi instanciado.
.hashCode(): exibe um código de hashing como se fosse uma identificação exclusiva.
.toString():exibe uma mensagem na forma de string com o nome da classe e alguns números.
