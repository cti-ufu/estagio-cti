Uma forma de fazer um melhor controle, é usando o método sleep(). Ele serve para que você faça o thread que está no estado atual de running sair deste estado, dando a oportunidade para outro thread de dar um run().

Este método garante que o thread que entrou no estado de sleep não saia dele antes de que o tempo do sleep() tenha expirado. Por exemplo, se definir que um thread fique no estado sleep por 2 segundos (2000 milissegundos), este mesmo thread nunca voltará ao estado de running até que os 2 segundos tenha passado.
