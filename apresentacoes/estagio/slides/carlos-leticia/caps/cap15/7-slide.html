As portas de 0 até 1023 são reservadas para aplicativos conhecidos (HTTP, HTTPS, POP3, etc), não sendo permitido utilizá-los para representar qualquer outro serviço, sendo permitido apenas um programa por porta.

Para representar os programas próprios que você criou, pode-se utilizar as portas de 1024 até 65535.

Quando for escrever um programa de servidor para “rodar” em uma rede de empresa, será necessário checar com os administradores de sistemas para saber qual das portas já estão sendo utilizadas, para que você não use utilize-as.
