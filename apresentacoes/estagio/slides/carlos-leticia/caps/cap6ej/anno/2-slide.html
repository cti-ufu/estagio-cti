Annotations podem ter elementos os quais são possíveis de terem valores inseridos. Um elemento é como um atributo ou paramêtro.
		// Annotate a method.

@MyAnno(str = "Annotation Example", val = 100)

public static void myMeth() { // …
Neste exemplo, há dois elementos com valores, separados por vírgulas: str e val.
Se um annotation conter apenas 1 elemento, por questão de padrões de nomeação, ele deverá ser chamado de value.
Ele pode ter seu elemento declarado com o seu nome, ou sem o seu nome (caso seja apenas um elemento value):
		@InsertNew(value = "yes")  //OU
@InsertNew("yes")
