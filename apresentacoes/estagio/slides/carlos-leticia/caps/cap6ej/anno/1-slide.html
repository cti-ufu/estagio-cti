Annotations
Java Annotations são usados para prover dados para os seus dados (metadados), sendo informações que podem ser vistas durante a compilação do programa.
Eles não afetam diretamente a execução do seu código, apesar que alguns tipos de annotations podem ser usados para esse propósito.
Consistem em um conjunto fixo de constantes, ou seja, static final, sendo como uma lista de valores pré-definidos.
Propósitos:
  • Instruções do compilador;
  • Intruções do Build-Time (Quando você constrói seu projeto de software);
  • Instruções do Tempo de Execução;
