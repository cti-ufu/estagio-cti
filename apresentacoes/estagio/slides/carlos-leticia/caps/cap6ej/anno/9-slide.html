@Inherited
Serve para definir que um annotation será herdado para as subclasses da classe que fará uso deste annotation.
java.lang.annotation.Inherited

@Inherited
public @interface MyAnnotation {

}

@MyAnnotation
public class MySuperClass { ... }
public class MySubClass extends MySuperClass { ... }
MySuperClass utiliza o @MyAnnotation, que foi marcado como @Inherited, fazendo com que o MySubClass (que é uma subclasse do MySuperClass) também herdará o @MyAnnotation.
