@Documented
O annotation @Documented é usado para sinalizar para a ferramenta JavaDoc (gerador de documento do Java, gerando as documentações em formato HTML do código Java) que seu annotation deve estar visível na JavaDoc para as classes que utilizam seu annotation.
Aqui está um exemplo de anotação @Documented Java:
import java.lang.annotation.Documented;

@Documented
public @interface MyAnnotation {

}
@MyAnnotation
public class MySuperClass { ... }
Quando gerar um JavaDoc para a classe MySuperClass, o @MyAnnotation será incluso.
