Criando seu próprio Annotation
Os Annotations são definidos em seus próprios arquivos, assim como uma class Java.

// A simple annotation type.
@interface MyAnno {
	String str();
	int val();
}
Para definir que é um annotation, será usado a palavra chave @interface (annotations são interfaces).
Pode definir dentro de seu corpo elementos primitivos e métodos (vazios, por ser uma interface), não podendo usar objetos como tipo de dado.
Dentro deles, podem ser definidos valores padrões para os elementos, que irão ser utilizados caso não tenha sido definido valores no parentesis da chamada do Annotation.
