Built-in Java Annotations
O Java já vem com 3 métodos que são usados para dar instruções ao seu compilador. Elas são:

@Deprecated
Usado para marcar uma classe, método ou campo como deprecated (obsoleto), significando que não será mais utilizado.
Se seus códigos utilizarem métodos marcados como deprecated, o compilador do Java lhe dará um aviso.
É bom utilizar este annotation para evitar que programas possam ter erros de compatibilidade, no meio do processo de implementação de uma nova classe, método ou campo.
Também é bom utilizar um comentário para explicar porque o método, classe ou campo está obsoleto e o que utilizar no lugar :
@Deprecated
/**
  @deprecated Use MyNewComponent instead.
*/
public class MyComponent {

}
