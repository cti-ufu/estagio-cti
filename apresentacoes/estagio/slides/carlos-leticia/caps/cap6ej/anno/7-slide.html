@Retention
Uma politica de retenção (@Retention) determina em que ponto um annotation é descartado.
Há 3 politicas encapsuladas na classe java.lang.annotation.RetentionPolicy :
  • SOURCE → O annotation é retido apenas no arquivo fonte e descartado durante a compilação;
  • CLASS → O annotation é guardado dentro do arquivo .class, porém não fica disponível durante a execução dessa classe;
  • RUNTIME → Um annotation com uma política de retenção de RUNTIME é armazenado no arquivo .class durante a compilação e está disponível através da JVM durante o tempo de execução. Assim , a retenção RUNTIME oferece a maior persistência de anotação.
    •  
    import java.lang.annotation.Retention;
import java.lang.annotation.RetentionPolicy;

@Retention(RetentionPolicy.RUNTIME)

@interface MyAnnotation {

    String   value() default "";

}
