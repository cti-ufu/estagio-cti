Métodos do Enum
  • values() → Retorna um array contendo uma lista das constantes do enum;
  • valuesOf() → Retorna a constante do enum o qual o seu valor corresponde a String declarada dentro do parênteses;
  • name() → Retorna o nome da constante do enum como String.
    • 
Apple allapples[] = Apple.values();

Apple ap = Apple.valueOf(“WINESAP”);

String nome = ap.name();

String nome2 = Apple.WINESAP.name();
