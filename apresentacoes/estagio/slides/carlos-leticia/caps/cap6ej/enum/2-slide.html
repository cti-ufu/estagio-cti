Enums
  • Os Enums são usados para a criação de estruturas de dados organizados, podendo agrupar valores que tenham o mesmo sentido para dentro dessa estrutura.
  • Consistem em um conjunto fixo de constantes, ou seja, static final, sendo como uma lista de valores pré-definidos.
    •
    Características:
  • O valor é fixo;
  • Não se pode criar uma instância de um ENUM;
  • Por serem objetos constantes e imutáveis, nomes declarados recebem todas as letras em MAIÚSCULAS;
  • Pode incluir variáveis de instância, construtor, métodos de instância, de classe, etc.
  • Eles podem ser públicos, mas é melhor torná-los privados e fornecer acessores públicos.
