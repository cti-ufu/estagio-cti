Comparable

. O método sort() só pode usar listas de objetos Comparaple.
. Ao classificar uma lista de objetos de outro tipo, ele não funcionará, mas funcionará ao classificar uma lista de Strings.
	public static < T extends Comparable  < ? super T> > void sort(List < T > list)
Para os tipos genéricos, "extends" significa "extends" ou "implements".
. Só podemos passar uma lista de objetos para o método sort() se a classe do Objeto implementar Comparable.
. Os documentos da API mostra que a interface Comparable possue apenas um método para implementar.

java.util.Comparable
