.Permite especificar parâmetros, tipos ou subtipos parametrizados.
.Uma maneira de criar um argumento de método que possa aceitar uma ArrayList de qualquer subtipo.
O método abaixo, só irá aceitar listas que extendam Number, ou seja, elementos das subclasses de Number.
 
	public double soma(ArrayList<? extends Number> lista){
		doble total = 0;
		for(Number : lista){
			total+=number.doubleValue();
		}
	}
Em caso de pais, ou seja, superclasses:
 
	public double soma(ArrayList < ? super Number> lista){
		doble total = 0;
		for(nomeSuperclasse : lista){
			total+=number.doubleValue();
		}
	}
