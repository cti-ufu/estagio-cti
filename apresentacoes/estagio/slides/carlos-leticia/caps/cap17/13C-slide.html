Java Web Start (JWS)

O Java Web Start inicia o aplicativo a partir de um navegador, porém executado como um aplicativo autônomo, não possuindo restrições do navegador.

Ele é uma versão pequena do Java, que funcionará de uma maneira semelhante a um plug-in de navegadores.

Sua principal função é gerenciar o download, atualização e execução de seus aplicativos Java.
