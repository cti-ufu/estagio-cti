Implementando um aplicativo
.Um programa Java é um conjunto de classes. Esse será o resultado do seu desenvolvimeno.
.Opções de implementação: local, combinação de local e remota, e remota.
.Separe os arquivos de código-fonte de classes, assegurando que os arquivos de classes compiladas não fossem inseridos no mesmo diretorio de código-fonte.
.Dica do livro: Cria um diretório do projeto e dentro dele criará um diretṕrio chamado source e outro de classes. use o flag do computador -d para que ao compilar o código a saida (.class) vá para a pasta classes.
% cd MyProject/source
%javac -d.../classes MyApp.java
.O comando *.java compilará todos os arquivos de código fonte do diretório atual.
