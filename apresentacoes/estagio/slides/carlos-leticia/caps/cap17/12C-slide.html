Caso o usuario não possua o JVM instalado em sua maquina, não será possível executar os códigos Java, incluindo o JAR.

Para resolver este problema, uma solução seria criar um instalador customizado do Java e distribuir ele junto de seu aplicativo. Assim, caso o usuario não possua o JVM, o instalador detectará isso e instalará o Java na máquina, permitindo que o seu aplicativo seja executavel nesta mesma máquina.
