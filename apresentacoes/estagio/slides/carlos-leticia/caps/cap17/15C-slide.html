Para obter o JWS, basta instalar o Java Runtime Environment (JRE) que ele vem instalado junto automaticamente.

É possível estabelecer (set) em sua página, que caso o usuário-final não possua o JWS, ele será baixado através do Sun.

Caso o usuário-final possua o JWS, mas desatualizado, ele levará o usuário a fazer o download do Java SE mais atualizado (conforme a versão que você esta utilizando).
