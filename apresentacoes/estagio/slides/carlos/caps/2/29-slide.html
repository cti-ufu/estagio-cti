Exceções são polimórficos

Exceções são objetos. E por serem objetos, é possível aproveitar do polimorfismo. Pode ser usado uma exceção superclasse para dar Catch em todas as subclasses de exceções dela, ou para especificar um catch para cada subclasse (ou conjunto de subclasses).

Apesar de poder usar um super Catch para pegar todas as exceções, é aconselhado fazer um catch especifico para cada exceção, a fim de ter a individualidade de recuperação para cada RuntimeException.
