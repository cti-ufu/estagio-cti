Math.round(num.f) → Retorna um valor do tipo INT ou FLOAT (dependendo se o argumento é do tipo FLOAT ou DOUBLE) arredondado para o valor mais próximo. (De 0.0 até 0.49, é arredondado para 0. De 0.5 até 0.99, será arredondado para 1).

Math.min(num1, num2) → Retorna o menor argumento dos dois que foram inseridos. (Pode ser int, long, float, double).

Math.max(num1, num2) → Retorna o maior argumento dos dois que foram inseridos. (Pode ser int, long, float, double).
