1º - Se seus dados forem usados apenas pelo programa Java que gerou eles: Usa-se serialização, em que será escrito um arquivo que vai conter objetos achatados (serializados). Então tenha o programa para ler os objetos serializados do arquivo, fazendo-os voltar a vida (inflar). O arquivo vem com textos que não farão sentido se for lido, porém é muito mais facil (e seguro) para seu programa restaurar o objeto da serialização.

2º - Se seus dados forem usados por outros programas: Digite um arquivo plano de texto, com delimitadores que outros programas possam analizar. Os objetos são separados em linhas, separando os valores das variáveis de instância através de vírgulas.
