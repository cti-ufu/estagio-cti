Empacotamento/Desempacotamento APÓS o Java 5.0

Do Java 5.0 para frente, é feito automaticamente o empacotamento (Autoboxing). Apenas basta definir o tipo de objeto empacotador (Integer, Boolean, etc) na classe de coleção (ArrayList e HashMap). Lembrando que é apenas possível colocar classes como tipo do ArrayList e HashMap, não podendo colocar int, por exemplo, dentro dos colchetes.

Exemplo: 
	
		ArrayList listOfNumbers = new ArrayList();
		
		listOfNumbers.add(1); //Automaticamente empacotou o primitivo 1 
		
		int num = listOfNumbers.get(0); //E automaticamente desempacotou o primtivo 1, que encontrava-se no index 0
