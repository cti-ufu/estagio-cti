Geralmente o Autoboxing é feito passando uma referência para o empacotador ou um primitivo do tipo correspondente.

Ele é feito nas seguintes situações:

  • Argumentos de métodos
  • Valores do return
  • Expressões booleanas Também pode utilizar uma expressão de comparação que resulte em uma booleana
  • Operações sobre números
    • Ex: Integer j = new Integer(5); Integer k = j + 3;
  • Atribuições
    • Ex: int x = 3; Integer y = x; OU Integer y = 3; int x = y;
