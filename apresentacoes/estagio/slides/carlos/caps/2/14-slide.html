Formatação de Números

Utilizando o format

Ex: String s = String.format(“%,d”, 1000000); // 1,000,000

% [número de argumento] [flags] [mínimo] [.precisão] tipo

Número de argumento → Representa o index dos argumentos passados para os métodos do format ou o resultado array retornado para os métodos parse.

flags → Formatações especiais, como inserir vírgulas, colocar números negativos em parênteses, etc.

Mínimo → quantidade mínima de caracteres que pode ser colocado no especificador.
