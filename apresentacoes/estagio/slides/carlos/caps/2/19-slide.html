Static Import

Static import serve apenas para digitar menos os métodos de classes static, para facilitar na digitação. O problema é que isso pode tornar a leitura dos códigos um pouco mais difícil, caso tenha classes static diferentes com o mesmo nome de método.

Exemplos: 
	
		Normal:
		import Java.lang.Math;
		System.out.println(“sqrt “ + Math.sqrt(2.0));

		Com Static:
		import static Java.lang.Math.*;
		import static Java.lang.System.out;

		out.println(“sqrt “ + sqrt(2.0));
