Movimentação de Dados em Streams

O API de E/S do Java tem Fluxos de Conexão, que representam as conexões com destinos e origens, tais como arquivos e soquetes de rede. E Fluxos de Cadeia só funcionam quando encadeados a outros fluxos.

Normalmente é utilizado dois fluxos ligados entre si para tomar uma ação. Um para representar a conexão e o outro para chamar métodos.
