Classe Exemplo

Um exemplo de uma classe que encontra-se no Java API, o qual poderemos utilizar em muitos programas e classes que construiremos ao longo de nossa carreira, seria a classe ArrayList.

A seguir, alguns métodos desta classe: 

	
add(Object elem)  //Adiciona o parametro do objeto para a lista
remove(int Index) //Remove o objeto no parametro Index
remove(Object elem) //Remove este objeto (se ele estiver no ArrayList)
contains(Object elem) //Retorna 'true' se há algum equivalente para o   parametro do objeto
isEmpty() //Retorna 'true' se não há elementos na lista
indexOf(Object elem) //Retorna o index do parametro do objeto, ou -1
size() // Retorna o atual número de elementos na lista
get(int index) //Retorna o objeto que encontra-se no parametro do index
