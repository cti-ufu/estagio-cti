Objetos Array (Matriz)

São objetos que podem definir uma quantidade de células que podem ser usadas para guardar variáveis ou objetos

Exemplo: 
	
		Dog[] pet = new Dog[7];

		pet[0] = new Dog();
		pet[1] = new Dog();

		pet[0].setSize(30);
		int x = pet[1].getSize(); //Irá retornar 0, pois quando não for definido um valor, poderá retornar como 0 (primitivas), False (boolean) ou Null (Objetos)
