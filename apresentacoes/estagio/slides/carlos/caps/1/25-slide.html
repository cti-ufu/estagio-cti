Como usar a biblioteca (Java API)

Classes são agrupadas em pacotes, e para usar a classe que você quer, precisa saber em qual pacote ela se encontra.

O pacote Java.lang (String, Math.random(), System, etc) já é incluso automaticamente

Para chamar uma classe de um pacote: import NomeDoPacote.NomeCompletoDaClasse;

Ex: import Java.util.ArrayList;

Se não for feito o import, terá que digitar o nome do pacote e o nome da classe, toda vez que for querer utilizar ela

Ex: java.util.ArrayList list = new java.util.ArrayList ();
